MPNativeAdRendering Protocol Reference

Conforms to NSObject
Declared in MPNativeAdRendering.h

Overview

The MPNativeAdRendering protocol provides methods for displaying ad content in custom view classes.

Tasks

  • – layoutAdAssets: required method
  • + sizeWithMaximumWidth:

Class Methods

nibForAd

Specifies a nib object containing a view that should be used to render ads.

+ (UINib *)nibForAd

Return Value

an initialized UINib object. This is not allowed to be nil.

Discussion

If you want to use a nib object to render ads, you must implement this method.

layoutAdAssets:

Populates a view’s relevant subviews with ad content.

- (void)layoutAdAssets:(MPNativeAd *)adObject

Parameters

adObject

An object containing ad assets (text, images) which may be loaded into appropriate subviews (UILabel, UIImageView) via convenience methods.
